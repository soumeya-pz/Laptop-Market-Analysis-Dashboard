## 🛠️ Project Implementation (Step-by-Step)

### 1. Data Cleaning & Transformation (Power Query)
Before visualization, the raw data underwent a rigorous ETL process:
* **Unit Normalization:** Stripped units like "GB" and "kg" to convert technical specs into clean numeric values.
* **Data Validation:** Handled missing values and ensured correct data types for mathematical operations.
* **Feature Engineering:** Segmented data by Brands and Laptop Types for deeper analysis.

### 2. Analytical Architecture (Pivot & Logic)
* **KPI Engineering:** Calculated vital market metrics: Total Inventory, Max Price, and Average Pricing.
* **Dynamic Interactivity:** Integrated Slicers for Brands and Segments, allowing for real-time cross-filtering.
* **Trend Analysis:** Created visualizations for OS distribution and RAM configurations.

### 3. UI/UX Design & Aesthetics (The "Pro" Look)
* **Custom Background:** Designed a bespoke, dark-themed background to create a structured visual hierarchy.
* **Modern Interface:** Used custom shapes and layout alignment to separate metrics from charts, reducing visual clutter.
* **Professional Typography:** Unified the color palette and fonts for a consistent, corporate-ready aesthetic.

## 💡 Key Market Insights
* **8GB RAM** remains the standard market configuration across most brands.
* **Notebooks** lead in volume, while high-performance segments are dominated by specific premium brands.
* **Pricing Strategy:** Clear distinction observed between budget-friendly volume leaders and premium-tier innovators.
